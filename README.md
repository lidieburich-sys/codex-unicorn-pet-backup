# Codex Unicorn Pet Backup

Codex 桌宠独角兽备份。

## 目录

- `source/`: iCloud 里的完整制作文件和预览资源
- `installed/`: 当前安装在 `~/.codex/pets/codex-unicorn` 的实际运行资源

## 恢复安装

如果只需要恢复当前桌宠资源，可以把 `installed/` 里的文件复制回：

```bash
~/.codex/pets/codex-unicorn/
```

如果需要重新构建贴图或修改动作资源，看 `source/` 目录。

## 说明

本仓库只备份桌宠资源和脚本，不包含系统缓存文件。
